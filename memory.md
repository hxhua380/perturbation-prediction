## 2. 长期研究目标

- 研究 AIVC（AI Virtual Cell，人工智能虚拟细胞）中的 Single-cell perturbation prediction（单细胞扰动预测）：根据未处理细胞及扰动条件，预测扰动后的表达状态或分布。
- 选择成熟、公开、可复现的 Baseline（基线模型）：用来衡量后续改进是否有效的原始模型。
- 在现有模型上完成工作量可控、逻辑合理、可实验验证的增量改进。
- 目标成果为普通期刊、中文核心、一般 SCI 或 CCF-C 级别论文，不以大型基础模型或顶级论文为目标。
- 最终形成可复现代码仓库、完整实验记录和论文材料。

## 3. 当前研究范围

- 单基因与多基因组合扰动、CRISPR 基因扰动、药物扰动。
- 未见扰动预测、跨细胞类型/跨数据集泛化、分布级预测、时间动态预测。
- 基础模型、Gene embedding（基因嵌入）：把基因转换为模型可处理的数字向量；Biological prior（生物学先验）：模型外已有的生物学知识。
- OOD generalization（分布外泛化）：面对训练分布之外的数据仍能有效预测。
- 评估指标、数据划分、数据泄漏、鲁棒性和生物学解释。

## 4. 不希望采用的研究路线

- 不从零训练新的大型基础模型。
- 不为了“创新”堆砌复杂模块，不把简单拼接包装成重大理论创新。
- 不优先选择代码不可得、数据受限、环境极难复原或训练成本不可控的方法。

- 不同时复现许多大型模型；最终聚焦 1 个主 baseline、1 个备用 baseline、1 个简化模型。
